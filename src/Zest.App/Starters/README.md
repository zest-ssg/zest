# Zest Blog

A starter blog generated by `zest init`. It demonstrates Zest's **native**
template mode: pages are written in `.zest.fsx` (real F# code) and HTML
layouts are automatically processed by the Nunjucks engine, while styles use
ZCSS (`.zcss`).

## Structure

```
.
├── _config.toml            # Site config (native template engine)
├── _init.zest.fsx          # Runs before each build (injects globals)
├── _layouts/
│   ├── default.html        # Base layout (Nunjucks)
│   └── post.html           # Article layout (Nunjucks)
├── _includes/
│   ├── header.html         # Site header / nav
│   └── footer.html         # Site footer
├── content/
│   ├── index.zest.fsx      # Home page
│   ├── about.zest.fsx      # About page
│   └── posts/
│       ├── index.zest.fsx  # Blog archive
│       ├── welcome-to-zest.zest.fsx
│       └── building-sites-with-fsharp.zest.fsx
└── assets/
    └── css/
        └── main.zcss       # ZCSS → compiled to main.css
```

## Commands

```bash
zest serve     # Build + live-reload dev server (http://localhost:8080)
zest build     # Build the static site into _site/
zest preview   # Serve the built _site/ without rebuilding
```

Edit any file and the dev server rebuilds instantly.
